SVG 단축어란

SVG 단축어는 휴대폰 화면, 문자 메시지, SNS 알림 등을 실제 이미지처럼 렌더링해주는 기능입니다. 단축어를 입력하면 외부 서버에서 SVG 코드를 생성하고, 그 결과가 채팅창에 이미지 형태로 표시됩니다.

서버에 어떤 정보가 남나요

SVG가 생성되는 과정에서, 제작자의 서버에는 다음과 같은 정보가 기록될 수 있습니다.

  1. SVG를 생성하는 데 사용된 URL 및 파라미터 전문
  2. 요청 시각 및 타임존
  3. IP 주소
  4. ISP / 통신사 모바일은 통신사, 유선은 인터넷 사업자가 표시됩니다.
  5. 접속 기기 및 브라우저 정보
  6. 접속 환경 추정 (앱 / 웹) 기기 정보로부터 유추되는 값으로, 정확하지 않을 수 있습니다.
  7. 접속 국가
  8. 접속 지역 및 도시 이하 8·9·10번은 IP 기반 추정값으로, 실제 위치와 다를 수 있습니다.
  9. 우편번호
  10. 위도 / 경도

채팅 내용이 완성된 이미지째로 저장되는 것은 아닙니다. SVG를 만들기 위해 서버에 전달된 링크와 파라미터가 로그로 남는 구조입니다. 메시지 수정 버튼을 눌렀을 때 보이는 긴 URL이 바로 그 원본 형태입니다.

또한 한 번 생성된 SVG 기록은 채팅방을 삭제해도 사라지지 않습니다. 이미 외부 서버로 호출된 결과이기 때문입니다.

저는 로그를 수집하지 않습니다

현재 제 SVG 서버는 로그 수집 기능을 OFF 상태로 설정해 두었습니다. Cloudflare 인프라 자체에는 흔적이 남을 수 있지만, 제 관리자 대시보드에는 어떤 기록도 표시되지 않으며 저는 여러분의 정보를 확인할 수 없습니다.

오류가 발생할 때

SVG 단축어도 서버 상태나 모델 처리 과정에 따라 출력이 정상적으로 나오지 않을 때가 있습니다. 발견될 때마다 수정하고 있으며, 반복적으로 같은 오류가 발생하는 단축어가 있다면 작품 댓글이나 가이드 글 댓글로 알려주시면 감사하겠습니다.

자주 묻는 질문

수정을 눌렀을 때 알파벳 링크가 길게 나오는 경우 — 생성 오류입니다. 해당 메시지를 삭제하고 단축어를 다시 입력해 보세요. 모델은 프로챗 2.5 또는 하이퍼챗을 권장합니다.

링크가 한글 포함으로 정상 출력되었는데 이미지가 사라지는 경우 — 메시지 하단 ··· 메뉴에서 '이미지 상황 보기'를 켜주세요.

링크가 중간에 잘리는 경우 — 삭제 후 재전송하거나, '이어서 생성'으로 나머지 링크를 받아 이어 붙여 주세요.
